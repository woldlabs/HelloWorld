# HelloWorld

A simple Hello World application built for WoldLabs.

![Hello World](https://img.shields.io/badge/Hello-World-brightgreen)

## Overview

This repository contains minimal, dependency-free Hello World examples in Python (CLI) and a modern web app (pure HTML/CSS/JS).

## Running the Apps

### Python CLI

```bash
python hello.py
```

Or on Windows:

```powershell
python hello.py
```

Expected output:

```
Hello, World! from WoldLabs
Welcome to the world of code.
```

### Web App

1. Open `index.html` directly in any modern browser (double-click the file).
2. Or serve it locally:
   - Python: `python -m http.server 8000`
   - Then visit http://localhost:8000/index.html

The page features an interactive greeting, theme toggle, and fun animations.
